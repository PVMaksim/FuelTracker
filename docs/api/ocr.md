# OCR API — Распознавание чеков

## POST /ocr/receipt

Принимает фото чека АЗС и возвращает распознанные данные через Claude API.

```http
POST /api/v1/ocr/receipt
X-API-Key: your-api-key
Content-Type: multipart/form-data

file: <image/jpeg или image/png, максимум 10MB>
```

**Ответ `200`:**
```json
{
  "total_cost": 2340.00,
  "liters": 40.00,
  "price_per_liter": 58.50,
  "station": "Лукойл",
  "raw_response": "{\"total_cost\": 2340.00, ...}"
}
```

!!! tip "Если поле не распознано"
    Нераспознанные поля возвращаются как `null`.  
    Пользователь всегда может скорректировать значения вручную перед сохранением.

**Ошибка `503`:** `ANTHROPIC_API_KEY` не настроен в `.env`

---

## Промпт для Claude

```python
OCR_PROMPT = """
Ты анализируешь фотографию чека с автозаправочной станции.
Найди и верни ТОЛЬКО JSON без пояснений:

{
  "total_cost": <итоговая сумма в рублях>,
  "liters": <количество литров>,
  "price_per_liter": <цена за литр>,
  "station": "<название АЗС или null>"
}
"""
```

## Поддерживаемые форматы чеков

Тестировалось на чеках:
- Лукойл
- Газпромнефть  
- Роснефть (Башнефть)
- Татнефть

!!! note "Качество распознавания"
    Убедись что:

    - Чек хорошо освещён
    - Текст в фокусе
    - Нет сильного засвета/тени
